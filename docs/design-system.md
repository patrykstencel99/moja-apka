# PatternFinder Design System

## Theme
- Theme key: `pf-premium`.
- Styl: calm high-density.
- Kompozycja: 80% neutral, 15% gold/silver, 5% color accent.

## Palette
- Base: `#0A0A0B`, `#F6F6F3`, `#1C1F26`, `#B9BEC8`.
- Premium accent: `#C8A35D`.
- Semantic accents: `#5B1F2A`, `#1E3FAE`, `#5B3A75`.

## Typography
- UI: Plus Jakarta Sans.
- Display: Sora.
- Data/mono: IBM Plex Mono.

## Tokens
- Source files:
  - `src/design/tokens.css`
  - `src/design/semantic.css`
- Groups: color, type, space, radius, shadow, motion, layout.

## Components
- Primitive components:
  - `Button`, `Card`, `StatTile`, `RangeField`, `CheckboxTile`, `Banner`.
- Rule: no hardcoded colors in screen components.

## Layout
- 12-col desktop, 6-col tablet, 4-col mobile.
- Spacing scale: 4/8/12/16/24/32/48/64.

## Motion
- Durations: 140ms / 220ms / 320ms.
- Effects: subtle fade+translate, minimal hover elevation.

## Accessibility
- Kontrast minimum AA.
- Wyrazne focus states.
- Czytelna hierarchia naglowkow i blokow danych.
